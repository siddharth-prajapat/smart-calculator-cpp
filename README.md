# Smart Calculator in C++

This is a **menu-driven Smart Calculator** developed in **C++**.
The program performs multiple mathematical operations and also saves the calculation history using **file handling**.

##  Features

* Addition
* Subtraction
* Multiplication
* Division
* Square
* Square Root
* Percentage Calculation
* History stored in a file
* Menu-driven interface

##  Technologies Used

* C++
* File Handling (`fstream`)
* Object Oriented Programming
* Console based UI

##  File Used

The calculator saves all operations in a file:

```
Calculate.txt
```

Example history:

```
Addition is :- 5+3=8
Subtraction is :- 10-4=6
Multiplication is :- 7*5=35
Square is :- 6^2=36
Square Root of 25 = 5
Percentage of 420 out of 500 = 84%
```

##  How to Run

1. Compile the program using a C++ compiler
2. Run the executable file
3. Select an operation from the menu
4. View history from the history option
